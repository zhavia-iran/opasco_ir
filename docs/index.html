توضیحات

با دنبال کردن این راهنما می توانید سفارشی سازی قالب Eduport را شروع کنید، این راهنما به شما کمک می کند تا به سرعت و با درک آسان بیشترین استفاده را از قالب HTML ما ببرید.

از شما برای خرید قالب Eduport سپاسگزاریم! قبل از شروع سفارشی سازی آن ، توصیه می شود مراحل ذکر شده در این راهنما را مطالعه کنید. ما اطلاعات لازم را ذکر کرده ایم و تمام دستورالعمل‌ها و جزئیاتی را که برای استفاده از این قالب نیاز دارید در اختیار شما قرار داده ایم.

      • js

        شامل تمامی فایل های مربوط به پلاگین های قالب

اقدامات لازم بعد از خرید

1. بررسی فایل ها

بعد از خرید قالب از سایت راست چین فایل .zip را extract کنید.

2. سفارشی سازی

اکنون می توانید سفارشی سازی وب سایت خود را بر اساس نیاز خود شروع کنید.

3. حذف فایل و پوشه های غیرضروری

پس از تکمیل وب سایت خود، باید فایل و پوشه غیر ضروری را حذف کنید تا حجم وب سایت کاهش یابد. لطفاً پوشه‌ها/فایل‌های مورد استفاده را برای اجرای وب‌سایت خود بررسی کنید:

  • assets/css: به فایل style.css (یا style-rtl.scss برای نسخه RTL) نیاز خواهید داشت.
  • assets/images: شما می توانید تمام فایل های غیر ضروری را از این پوشه حذف کنید.
  • assets/js: این پوشه/فایل را حذف نکنید.
  • assets/vendor: فقط پلاگین هایی از این پوشه را که در وب سایت خود استفاده کرده اید نگه دارید.
  • docs: برای نمایش وب سایت خود نیازی به پوشه docs ندارید و این پوشه جهت راهنمایی خریداران عزیز است.
