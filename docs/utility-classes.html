کلاس های کاربردی

برای توسعه سریعتر وب سایت و سازگاری با موبایل، فریم ورک Bootstrap شامل ده ها کلاس کاربردی برای نمایش، پنهان کردن، تراز کردن و فاصله گذاری محتوا است.

Bootstrap

ما کلاس های جدیدی را در قسمت اضافه کرده ایم تا روند توسعه شما را تسریع کنیم. لیست زیر را بررسی کنید:

نام کلاس توضیحات سایر ویژگی ها
text-primary-hover کلاس .text-primary-hover برای تغییر رنگ متن هنگام قرارگیری اشاره گر ماوس روی متن -
font-base اعمال فونت پیش فرض -
font-heading اعمال فونت پیش فرض برای عنوان -
grayscale اعمال فیلتر مقیاس خاکستری روی تصویر یا هر بخش دیگر -
zoom-hover بزرگنمایی هنگام قرارگیری اشاره گر ماوس با افکت -
up-hover بزرگنمایی هنگام قرارگیری اشاره گر ماوس با افکت -
zoom-x مدیریت میزان zoom با استفاده از کلاس های: .zoom-1
.zoom-2

.z-index-99
z-index-n9 تنظیم z-index بصورت منفی -
