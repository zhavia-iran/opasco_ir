نحوه استفاده:


<!-- Choice select -->
<select class="form-select js-choice" data-search-enabled="true" multiple data-remove-item-button="true" data-position="bottom" data-max-item-count="3" data-placeholder="true">
  <option value="PB">عمومی</option>
  <option value="PV">مخاطبین</option>
  <option value="PV">خصوصی</option>
  <option value="PV">سفارشی</option>
</select>

<!-- Choice input -->
<input class="form-control js-choice" type="text" data-placeholder="true" data-placeholder-val="Custom placeholder value" data-remove-item-button="true" data-max-item-count="2">

ویژگی های دیگر این قابلیت به همراه توضیحات آن در جدول زیر ارائه شده است:

نام Attributes توضیحات سایر ویژگی ها
data-search-enabled="true" نمایش box جستجو Boolean (true or false)
data-position="bottom" موقعیت بازشدن dropdown (لیست کشویی) "top" | "bottom"
data-max-item-count="3" تعیین حداکثر انتخاب آیتم Numbers (1 2 3 4 5 6 ....)
multiple افزودن multiple به تگ <select>
data-remove-item-button="true" اینکه آیا هر مورد باید دکمه حذف داشته باشد یا خیر Boolean (true or false)
data-placeholder="true" فعال یا غیرفعال کردن placeholder برای <select> Boolean (true or false)
data-placeholder-val="انتخاب گزینه انتخابی" تعیین مقدار placeholder نوع ورودی String
